# AI Agent Architecture of Artemis II Mission

## Objective 
The objective of this essay was to analyze the Artemis II mission using Artificial Intelligence agent concepts and classify the different systems involved according to their agent architectures. The project focused on understanding how intelligent agents interact within a complex real world system and how different architectures contribute to mission success.

## Summary
The essay examined the Artemis II missions through the perspective of intelligent agents. Various components of the mission were analyzed and classified using agent architectures such as simple reflex, model, goal, utility, and human agents. We also explored how these systems interact through perceptions, decision making, and action while operating within a highly complex and safety environment. 

The analysis demonstrated that multiple agent architectures can coexist within a single system, with each architecture serving a specific role. While some systems are responsible for monitoring conditions and reacting to changes, others focus on achieving mission objectives, optimizing performance, or supporting human decision making.
